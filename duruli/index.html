흩어진 장면과 플롯을 적어두면, AI가 이야기를 노드 그래프로 그려줍니다.

두루리는 소설·웹소설 작가를 위한 iPad 이야기 구조 정리 도구입니다. 사건을 한 줄씩 적으면 AI가 장면 노드와 연결선을 순서대로 배치해, 이야기의 뼈대를 한눈에 보여줍니다.

자주 묻는 질문

AI 기능을 쓰려면 무엇이 필요한가요?
본인의 OpenAI API 키가 필요합니다. 설정 화면에서 키를 입력하면 AI가 이야기를 노드로 그려줍니다. 키 없이도 예시 보드는 둘러볼 수 있습니다.
API 키는 안전하게 보관되나요?
입력한 키는 이 기기의 Keychain에만 저장되며, 외부 서버로 전송하거나 노출하지 않습니다.
어떤 기기에서 쓸 수 있나요?
iPad 가로 모드 전용으로 설계되어 있습니다.
'커피 후원'은 무엇인가요?
두루리는 무료입니다. 커피 후원은 순수한 응원이며, 후원해도 추가 기능이 열리거나 하지 않습니다.
내가 쓴 내용은 어디에 저장되나요?
보드와 노드는 기기 안에 저장됩니다. AI로 그래프를 만들 때만 입력한 텍스트가 OpenAI API로 전송되어 처리됩니다.
