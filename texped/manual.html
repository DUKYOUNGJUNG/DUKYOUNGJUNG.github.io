1. 장비 옵션

장비 등급

장비는 5단계 희귀도로 구분됩니다. 등급이 높을수록 접두사/접미사 옵션이 붙을 확률이 높습니다.

고급녹색접두사 50% / 접미사 50% (서로 독립)
희귀파랑접두사 + 접미사 둘 다 항상 붙음
영웅보라접두사 + 접미사 둘 다 항상 붙음 (수치 상승)

트럼프 마크 옵션 (영웅 등급 전용)

영웅(보라) 등급 장비는 일반 접두사/접미사 외에 이름 끝에 트럼프 마크 + 숫자(1~5)가 무작위로 추가로 붙습니다. 각 마크는 고유한 특수 효과를 의미하며, 숫자가 클수록 효과가 강합니다.

마크의미효과 (N = 1~5)
♠스페이드 — 방어 무시적 방어력을 추가로 +N 차감하고 피해 계산 (접미사 ♠와 합산)
♥하트 — 생명력 흡수입힌 피해의 +N%를 자신 HP로 회복 (접미사 ♥와 합산, 전체 상한 30%)
♣클럽 — 크리티컬 확률치명타 확률 +N% (LUK 기반 확률에 가산, 전체 상한 50%)
♦다이아 — 크리티컬 피해치명타 발동 시 피해 배수 +N% 추가 (기본 1.5배 → 최대 1.55배)
중첩: 여러 영웅 장비를 함께 장착하면 같은 마크 효과가 합산됩니다. 동일 마크 위주로 묶어 빌드를 짜면 시너지가 강해집니다. (예: ♦5 × 4개 → 크리티컬 1.7배)

크리티컬 (LUK 기반)

크리티컬은 장비 옵션이 아니라 LUK 스탯으로 결정됩니다.

  • 치명타 확률 = LUK × 0.6% (최대 50%)
  • 치명타 피해 = 1.5배 고정
  • 마력각인(전투 — 크리티컬 +5%)으로 추가 가능
  • 일부 스킬은 자체 크리티컬 보너스 보유

상태이상

상태이상은 일반 장비 옵션이 아니라 전설 효과 또는 스킬/마력각인으로 부여됩니다.

이름효과
출혈매 턴 도트 데미지. 적 공격력 감소 디버프 동반. 누적 가능
중독매 턴 적 현재 HP의 10% 도트 데미지 + 적 DEF 50% 감소 (3턴)
화상매 턴 고정 데미지 + 회복 효율 감소
빙결적 행동 봉쇄. 보스는 보통 면역이며 대체로 약화 디버프 부여
기절다음 턴 적 공격 무효화

전설 효과 (전설 등급 전용)

전설 등급 장비는 고유 이름과 강력한 효과를 보유합니다. 도감에서 발견한 전설을 확인할 수 있습니다.

이름효과
심연의 이빨적 처치 시 누적 카운터 +1. 5 카운트마다 ATK +1. 20 카운트 이상에서 공격 시 출혈 부여
뇌신의 손길5회 공격마다 뇌격 발동 (250% 데미지 + 마비 1턴)
얼어붙은 영혼피격 시 10% 확률로 적 2턴 빙결 (빙결 면역 적은 약화 부여)
죽음의 속삭임전투 시작 시 HP 15% 헌납. 적 처치 시 MP +10 회복
별빛의 축복HP 30% 이하 시 최대 HP의 25% 회복 + 상태이상 해제 (전투당 1회)
월광의 파편7라운드 주기. 1-3R ATK/DEF +5%, 4-6R +10%/CRIT +5%, 7R HP 30% 회복
깊은 바다의 왕관층수가 높을수록 회피 증가 + 함정 피해 감소. 고층에서 선공 + 필중 크리
수정의 심장사망 시 심연석 3개 소모하여 부활 (HP 50% + 피해감소 50% 3턴). 런당 1회

전용 장비 (직업 전용무기)

일부 직업은 마을의 제련소에서 자신만의 전용 무기를 만들 수 있습니다.

  • 제련소에서 심연석으로 제작하고, 강화·진화로 성장시킬 수 있습니다 (일반 → 전설).
  • 전용 무기는 탐사가 끝나도 사라지지 않고 다음 탐사에 자동 장착됩니다.
  • 장착하면 스킬을 하나 더 장착할 수 있고, 직업 고유의 특수 능력(패시브)이 깨어납니다.

장비 코팅 (영속화)

장비에 특수 코팅을 입히면 사망 · 탐사 종료에도 사라지지 않습니다.

  • 코팅한 장비는 다음 탐사 시작 시 부위별 최고 등급 1개씩 자동으로 장착됩니다.
  • 제련소(마을)에서 장착 중인 장비를 코팅하거나, 휴대용 코팅제로 탐사 중 발견한 장비를 즉석에서 코팅할 수 있습니다.
  • 코팅하지 않은 일반 장비는 탐사가 끝나면 소실됩니다.

2. 전투 시스템

자동 전투

탐사 중 모든 전투는 자동으로 진행됩니다. 결과를 보고 캠핑/장비 변경으로 빌드를 조정합니다.

피해 계산

  1. 명중 판정: 공격자 HIT vs 방어자 회피. 명중 실패 시 빗나감 (0 데미지)
  2. 크리티컬 판정: LUK × 0.6% (최대 50%). 발동 시 1.5배 데미지
  3. 피해 계산: 공격력 - (방어력 - 방어무시)
  4. 흡혈 적용: 생흡 % 만큼 자신 HP 회복 (상한 30%)
  5. 상태이상 적용: 출혈/중독/화상/빙결/기절 부여 (확률/조건부)

상태이상 규칙

  • 지속 시간: 일반적으로 2~3턴
  • 중첩: 출혈/중독은 누적 가능. 화상/빙결/기절은 보통 중첩 X
  • 보스 면역: 일부 보스는 빙결/기절 등에 면역. 대신 "약화" 디버프로 대체
  • 해제: 정화의 결정 또는 일부 전설 효과(별빛의 축복)로 해제

회피 / 방어

회피는 DEX 스탯 + 일부 전설 효과로 추가. DEF는 피해 자체를 감소시키는 별도 스탯입니다.

3. 마을 / 시설

마을은 5단계까지 발전합니다. 레벨업 시 신규 시설이 해금됩니다.

통화

  • 은화 — 탐사 통화. 탐사 중 처치 보상으로 얻으며, 캠핑의 떠돌이 상인에서 아이템·장비 구매에 씁니다. 마을로 돌아오면 0으로 사라집니다(탐사 한정).
  • 골드 — 마을 통화. 대장간 장비 구매, 도구상점 소비 아이템, 여관 침대 등 마을 거래에 씁니다. 영구 보관되며 환전소에서 심연석과 교환할 수 있습니다.
  • 심연석 — 핵심 자원. 대장간 리롤, 시설 해금, 신체강화, 스킬 구매 등에 씁니다. 영구 보관되며 환전소에서 골드와 교환할 수 있습니다.

마을 레벨

레벨이름해금 시설
1천막소생진, 환전소, 탐사봇, 제련소*
2개척지대장간, 도구상점, 도서관
3마을여관, 신체강화소
4도시스킬연구소, 광산, 지도 제작소
5요새마력각인소

* 제련소는 전용무기를 가진 직업(머스킷티어 · 몽크 · 로그 · 매직유저 · 음유시인)에게만 열립니다.

주요 시설

소생진

  • 탐사 시작 시 캐릭터 레벨 강화 (영구)
  • 사망/마을 복귀 시 설정된 시작 레벨로 다시 리셋
  • 기본 Lv.10까지, 진행도 달성 시 Lv.20까지 확장

환전소

  • 골드 ↔ 심연석 교환소
  • 심연석 → 골드 시세는 매일 바뀌며, 최근 7일 시세 그래프를 보고 타이밍을 잡을 수 있습니다
  • 골드 → 심연석은 고정 환율
  • 자주 이용하면 단골이 되어 더 유리한 교환율을 받습니다

대장간

  • 티어별 장비 제작 / 구매 (골드)
  • 리롤: 진열 장비를 새로 굴림. 누진 비용 (10/15/20/25/30 심연석)

도구상점

  • HP/MP 포션, 정화의 결정, 부활인형, ATK/DEF 부스트, 행운의 부적, 천사의 가호 등 소비 아이템 판매 (골드)
  • 해금 조건 충족 시 점차 상품 확장
단골 시스템: 대장간 · 도구상점 · 여관 · 환전소를 자주 이용하면 단골 등급이 올라(최대 Lv.10) 가격 할인과 더 유리한 교환율을 받습니다.

도서관

  • 도감 열람 — 몬스터/보스/장비/고양이 등 발견한 항목 확인
  • 고대언어학: 레벨업 시 탐사에서 마주치는 해독 불가 문헌을 읽을 수 있게 됨

여관

  • 탐사 시작 전 영구 버프 1회 적용 (다음 탐사 1회 한정)
  • 일반 침대 (골드): HP/MP +10%, 랜덤 스탯 +5
  • 고급 침대 (골드): 일반 효과 + 함정 무시 10%
  • 최고급 침대 (골드): 고급 효과 + 경험치 +10%

신체강화소

  • 4개 부위 영구 강화 (각 부위 최대 Lv.5)
  • 팔 → STR +2 / 레벨
  • 다리 → DEX +2 / 레벨
  • 몸통 → VIT +2 / 레벨
  • 머리 → LUK +2 / 레벨

제련소

  • 전용 무기 제작: 심연석 30으로 직업 전용무기를 만듭니다 (일반 등급 기본형)
  • 강화: 최대 +10, 강화 레벨마다 ATK 증가 (비용 15 + 20 × 레벨)
  • 진화: 강화 +10 + 레벨 조건 충족 시 상위 등급으로 (일반→고급 150, →희귀 300, →영웅 600, →전설 1200). 진화 후 강화는 초기화
  • 옵션 리롤: 전용무기의 접두/접미사 변경 (심연석 40, 고급 이상)
  • 장비 코팅: 장착 중인 일반 장비를 코팅해 영구 보존 (등급별 심연석 20~160)
음유시인은 삼지창 전용 제련만 제공되며, 일반 장비 코팅 기능은 없습니다.

스킬연구소

  • 심연석으로 직업별 스킬 영구 구매
  • 구매한 스킬을 스킬 슬롯에 장착해서 사용

광산

  • 광부 파견 — 4가지 등급 (일반 / 숙련 / 정예 / 전설)
  • 등급에 따라 파견 시간 + 채굴량 + 내구도 차이
  • 오프라인에서도 시간 진행. 위젯/노티로 완료 알림

지도 제작소

  • 단서를 사용해 탐사 층을 미리 해독
  • 지도로 제작한 층은 다음 탐사에서 스킵 가능

마력각인소

  • 3개의 각인 슬롯을 심연석으로 순차 해금 (100 / 250 / 350)
  • 슬롯마다 랜덤 능력이 부여되며 리롤로 갱신 가능 (잠긴 슬롯 수에 비례 누진)
  • 능력군: 전투(크리티컬·선공·흡혈·스킬 확률·회피·상태이상 부여) / 탐사(함정 감소·상자 보너스·맵 공개 등) / 경제(심연석/골드/XP 보너스, 상점 할인) / 생존(HP·MP 재생·부활·면역·시작 HP 보너스)

탐사봇 (T.F.W / 루카)

  • 최대 12시간 자동 탐사 (1, 2, 4, 8, 12시간 선택)
  • 시간 비례 심연석 보상
  • 취소 시 진행 시간 비례 보상에 30% 패널티 적용

4. 탐사 / 캠핑

층 진행

각 층은 일반 몹과 보스로 구성됩니다. 보스 처치 시 층 클리어 + 보상 + 캠핑 선택지가 열립니다.

캠핑

층 클리어 후 등장하는 휴식 단계. 다음 중 선택:

  • 장비 정리: 인벤토리 확인 + 장착 변경
  • 소비 아이템 사용: 부스트, 행운의 부적, 천사의 가호 등
  • 떠돌이 상인: 임시 상품 (장비, 소비 아이템) — 은화로 구매
  • 다음 층 진행: 곧장 다음 보스로

주요 소비 아이템

  • HP 포션 / MP 포션: 사용 시 즉시 사용이 아니라, 조건에 따라 전투 후 자동 회복 (HP 30% / MP 50%)
  • 행운의 부적: 사용 후 다음 1층 동안 장비 드랍률 2배 (도구상점 구매)
  • 천사의 가호: 사용 후 다음 보스 / 챔피언전에서 3턴간 받는 피해 무효 (도구상점 구매)
  • 정화의 결정: 해당 층 상태이상 면역
  • 부활 인형: 사망 시 자동 부활
  • ATK / DEF 부스트: 일정 층 동안 능력 강화
  • 휴대용 코팅제: 탐사 중 발견한 장비를 즉석에서 코팅해 영구 보존 (도구상점 구매, 심연석 150)
부스트 재적용: 행운의 부적 / ATK / DEF 부스트는 발동 중에는 다시 못 씁니다. 만료 후 재적용 가능.

마을 복귀 시 장비 소실

핵심 규칙: 탐사 도중 마을로 돌아오면 — 사망, 자발적 포기, 완주, 엔딩 — 코팅하지 않은 장비는 모두 소실됩니다.

코팅한 장비와 전용 무기는 예외로 사라지지 않고, 다음 탐사 시작 시 자동으로 다시 장착됩니다. 남기고 싶은 장비는 제련소에서 코팅하거나, 탐사 중이라면 휴대용 코팅제로 코팅해 두세요.

심연석 획득

  • 몹 처치: 소량
  • 보스 처치: 층별 보상 + 첫 클리어 보너스
  • 탐사 종료 시: 누적된 심연석을 마을 통장에 적립

심연석 캡 (저장 한도)

마을 레벨에 따라 보유 가능한 심연석 상한이 있습니다. 캡 초과 시 잉여분은 버려집니다. 마을 레벨업으로 캡 확장.

5. 기타 시스템

일일 접속 보상

매 5일 누적 접속마다 심연석 +100 자동 지급. 계정 단위(모든 슬롯 공유).

걸음 보상 (HealthKit 연동)

  • 건강 앱 데이터 연동 시 활성
  • 1,000보마다 심연석 +1 (하루 최대 10개)
  • 설정에서 권한 허용 필요

후원 (IAP)

  • 인앱 결제로 심연석 100개 즉시 지급
  • 1회성 결제 (영구 권리)
  • 결제 후 보상 누락 시 설정 → 후원 보상 복구 버튼으로 자가 복구

위젯

  • 광산 위젯: 광부 진행 상황 + 심연석 잔액 + 탐사봇 상태
  • 홈 화면에 추가해 한눈에 진행 확인

다이나믹 아일랜드 / Live Activity

마을 공사 진행 중일 때 다이나믹 아일랜드 + 잠금화면에 진행도 표시. 자동 활성.

알림 (Notification)

  • 광부 파견 완료
  • 탐사봇 완료
  • 마을 공사 완료
